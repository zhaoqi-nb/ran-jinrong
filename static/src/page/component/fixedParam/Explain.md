 ```
 {
    "params":{"content_name":"deliver_fee"},
    "effectElement":["648","649"]
}
 ```

## 2.需要配置参数

| 参数 | 是否必须 | 参考值 | 说明 |

| effectElement | 必须 | 示例 | 影响元素 |
